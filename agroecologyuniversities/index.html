The AgroEcology Domain in Costa Rica Data Analysis of Universities online libraries

For understand how is the relance of the agroecology in the country we must look into the universities and research center, to see how many papers, articles, books they are generating. Thats will give us a better view of the relance and the period of time in the country

In each university we look into the online repository, specially all the documents that fit with the query 'agroecologia'
Therefore on each file we look for the periods of time, language, type of file, common terms, geolocalization the terms, for make a network mapping.

The Tools used and how I did it?

For extract big ammount of data from repositories online, there is three way. First one the repository has the option to select multiple files and download the metadata, Second option is asking to the university the information and third scrap by yourself and your criteria the repository page
So for each each university I made different scraper, in some cases the repository requires of credentials and others no. In case of necesity I ask my friends of each university.
